# Katkıda Bulunma Rehberi

Teşekkürler! Katkıda bulunmak için lütfen aşağıdaki adımları takip edin:

1. Issue açın veya mevcut issue'yu alın.
2. Yeni bir branch oluşturun: `git checkout -b feature/isim`
3. Değişikliklerinizi yapın ve test edin.
4. PR açmadan önce kodu güncelleyin ve formatlayın.

Kod standartları ve test yönergeleri projede ileride belirtilecektir.
